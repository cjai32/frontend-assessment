Why is the result of ('b' + 'a' + + 'a' + 'a').toLowerCase() "banana"?

Answer : Because +'a' is the unary plus of the string "a", which is NaN. So we have "ba" + NaN + "a", which becomes the string "baNaNa" because NaN is converted to a string and then using toLowerCase() method, It is converting the string to lowercase letters. Therefore, We get "banana" as the answer.
